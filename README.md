# Employee Leave Tracker 🗓️

A full-stack web application for managing employee leave requests built with Spring Boot and MySQL.

## Features
- Employee and Manager roles
- Employees can apply for leave with reason and dates
- Managers can approve or reject pending leave requests
- View all leave history with status tracking

## Tech Stack
- Java, Spring Boot, Spring MVC
- MySQL, Spring Data JPA, JDBC
- Thymeleaf, HTML, CSS, Bootstrap 5
- REST architecture, MVC pattern

## How to Run
1. Install Java and MySQL
2. Create a database called `leavetracker` in MySQL
3. Update `application.properties` with your MySQL password
4. Run `./mvnw spring-boot:run`
5. Open `http://localhost:8080`
